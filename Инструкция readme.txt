4.	Реализуйте следующие API (5%):
a.	GET posts/

Проверка через URL: http://127.0.0.1:8000/posts/api/posts/

b.	GET posts/:id

Проверка через URL: http://127.0.0.1:8000/posts/api/posts/1/ (для поста с ID 1).

c.	POST posts/ (для POST запроса используйте Django Forms)

Можно через Django Forms или админ панель http://127.0.0.1:8000/admin/

d.	DELETE posts/:id/delete (после удаления пересылает на страницу с todos)

Проверка через URL: http://127.0.0.1:8000/posts/api/posts/1/delete/
